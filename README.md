# QEH 3D Print Hub

**一站式 3D 打印辦公室管理系統** — 從申請審批、案例追蹤、進度管理到物料庫存，全部在一個平台完成。

*All-in-one 3D printing office management system for Queen Elizabeth Hospital — from application & approval to case tracking, progress management & material inventory.*

## Quick Start

```bash
npm run dev        # Development server on http://localhost:8080
```

## Deploy

```bash
package.bat        # Build portable ZIP for hospital computers
```

See [WORKLOG.md](./WORKLOG.md) for development history.
