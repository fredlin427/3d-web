# QEH 3D Printing Office Manager — 工作日志

## 項目概述
為 QEH 3D 打印辦公室構建內部管理 Web 應用，用於管理日常記錄、案例進度、材料庫存、材料使用和統計。

## 技術棧
| 層級 | 技術 |
|------|------|
| 框架 | Next.js 16.2 (App Router) + TypeScript |
| 樣式 | Tailwind CSS v4 + shadcn/ui (base-ui) |
| 數據庫 | SQLite (via Prisma 7 + prisma-adapter-sqlite) |
| 圖表 | Recharts 3 |
| 表單 | react-hook-form + zod 4 |
| 通知 | Sonner |
| CSV | papaparse |
| 圖標 | Lucide React |

## 開發時間線

- **Phase 4**：案例管理模塊

- **Phase 5**：材料庫存管理
  - Material CRUD API + 列表頁 + 表單 + 詳情頁
  - 庫存警報邏輯（低庫存、即將過期、已過期）

- **Phase 6**：儀表板
  - 7 個統計卡片
  - 5 種圖表（柱狀圖、餅圖、折線圖）

  - CSV 導出

- **Phase 7**：庫存盤點
  - CSV 導出（庫存清單）

- **Phase 8**：報告

- **Phase 9**：種子數據

  - 主數據 + 進度步驟設置

## 數據庫設計
6 個表：
- `cases` — 案例記錄
- `case_progress_steps` — 進度步驟
- `materials` — 材料庫存
- `case_material_usage` — 材料使用記錄
- `stock_transactions` — 庫存交易歷史
- `settings` — 主數據配置
- `audit_logs` — 審計日誌

## API 端點
12 個 API 路由：
- `/api/cases` — CRUD
- `/api/cases/[id]/progress` — 進度步驟
- `/api/cases/[id]/duplicate` — 複製案例
- `/api/materials` — CRUD
- `/api/material-usage` — 材料使用記錄
- `/api/stock-transactions` — 庫存交易
- `/api/dashboard` — 儀表板統計
- `/api/stock-take/export` — 導出庫存清單
- `/api/stock-take/import` — 導入盤點數據
- `/api/reports` — 報告導出
- `/api/settings` — 設置 CRUD
- `/api/progress/[id]` — 進度步驟 CRUD

## 頁面路由
| 路由 | 描述 |
|------|------|
| `/` | 儀表板（統計卡片 + 圖表） |
| `/cases` | 案例列表 |
| `/cases/new` | 新建案例 |
| `/cases/[id]` | 案例詳情（進度 + 材料使用 + 審計日誌） |
| `/cases/[id]/edit` | 編輯案例 |
| `/materials` | 材料列表 |
| `/materials/new` | 新建材料 |
| `/materials/[id]` | 材料詳情（使用記錄 + 交易記錄） |
| `/materials/[id]/edit` | 編輯材料 |
| `/stock-take` | 庫存盤點（導出 + 導入） |
| `/reports` | 報告導出 |
| `/settings` | 主數據管理 |

## 運行方式
```bash
cd qeh-3d-print
npm install
npx prisma generate
npx prisma migrate dev
npx tsx prisma/seed.ts
npm run dev
```
